【Csharp】3.构造函数初始化语句

分类于 C#

 

如果希望派生类使用一个指定的基类构造函数而不是无参数构造函数，必须在构造函数初始化语句中指定它。

  • 使用关键字 base 并指明使用哪一个基类构造函数。
  • 使用关键字 this 并指明应该使用当前类的哪一个构造函数。

构造函数初始化语句由关键字 base 和要调用的基类构造函数的参数列表构成。使用关键字 base 时，基类构造函数初始化语句放在冒号后面，跟在类的构造函数声明的参数列表后面。

使用关键字 this 时，构造函数初始化语句可以让构造过程（实际上是编译器）使用当前类中其他的构造函数。这种语法很有用的另一种情况是，一个类有好几个构造函数，并且它们都需要在对象构造的过程开始时执行一些公共代码，对于这种情况，可以把公共代码提取出来作为一个构造函数，被其他所有的构造函数用作构造函数初始化语句。由于减少了重复代码，实际上这也是推荐的做法。

有时候在实例化对象时无法完全初始化对象，此时，必须禁止从类的外部调用构造函数，因为那样的话只会初始化对象的一部分，要避免这个问题，可以把构造函数声明为 private，而不是 public，然后只让其他构造函数使用它、
